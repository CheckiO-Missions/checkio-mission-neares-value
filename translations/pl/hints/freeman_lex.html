Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie?

Proszę bardzo:

def nearest_value(values: set[int], one: int) -> int:

    min_dist, closest_num  = None, None
    for value in values:
        dist = abs(one - value)
        if min_dist == None or min_dist> dist:
            min_dist, closest_num = dist, value
        elif min_dist == dist:
            closest_num = min(value, closest_num)
        
    return closest_num

Znacznie krótsze rozwiązanie, w którym można znaleźć minimalną wartość w zbiorze za pomocą dwóch parametrów: pierwszy to odległość (różnica bezwzględna) od liczby wejściowej, drugi: mniejsza wartość między dwiema liczbami w tej samej odległości 

def nearest_value(values: set[int], one: int) -> int:

    return min(values, key=lambda n: (abs(one - n), n))

W tym rozwiązaniu zaczynamy od one i zwiększając i zmniejszając sprawdzamy, czy nowa wartość znajduje się w zbiorze. 

def nearest_value(values: set[int], one: int) -> int:

    n = 0
    while True:
        if (one - n) in values:
            return one - n
        if (one + n) in values:
            return one + n
        n += 1
